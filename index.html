Sobre o aplicativo

O PrecificaFood ajuda empreendedores da alimentação a calcular preços de venda, organizar ingredientes, salvar produtos no cardápio, simular promoções e manter informações básicas do negócio.

Como usar

  • No menu Precificando, preencha o nome do prato, custos, taxas e margem de lucro.
  • Use a Geladeira para salvar ingredientes e reutilizar preços.
  • Salve produtos para montar seu Cardápio.
  • Veja sugestões no menu Promoções.
  • No Perfil, atualize nome, WhatsApp e foto da empresa.

Privacidade e dados

As informações preenchidas no app são usadas para funcionamento das ferramentas de precificação e organização. Dados salvos localmente podem ser removidos ao usar “Desconectar” no Perfil ou ao limpar os dados do aplicativo no dispositivo.

Problemas comuns

  • Se algum valor não calcular, confira se os campos numéricos foram preenchidos corretamente.
  • Se dados sumirem, verifique se o armazenamento do navegador/app foi limpo.
  • Se a foto não aparecer, tente selecionar outra imagem do dispositivo.
  • Para relatar erro, envie detalhes e prints para o e-mail de suporte.

Privacidade e Termos

Resumo de privacidade

O PrecificaFood salva dados digitados pelo usuário no próprio dispositivo para permitir o funcionamento do app, como nome da empresa, foto, produtos, ingredientes, configurações e rascunhos.

Coleta e compartilhamento

Esta versão não envia dados para servidor próprio e não compartilha informações com terceiros pelo código interno do app. Caso seja adicionado serviço externo no futuro, a política deverá ser atualizada.

Permissões

O app pode usar seleção de imagem quando o usuário decide adicionar foto. Não solicita contatos, localização, SMS, microfone ou permissões sensíveis para funcionar.

Exclusão de dados

O usuário pode remover dados apagando itens individualmente, usando a função de sair/desconectar ou limpando os dados do app no dispositivo.
